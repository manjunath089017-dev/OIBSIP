Lostify

LostLink is a web-based application designed to manage lost and found items efficiently through a centralized digital platform. The system allows users to register, report lost or found items, and search existing records easily. It reduces manual paperwork and improves transparency in tracking items. The admin dashboard enables monitoring of entries, user management, and item matching. Developed using HTML, CSS, javascript and MySQL, the system ensures secure and organized data handling. LostLink aims to simplify item recovery, enhance communication between users and administrators, and provide a reliable solution for institutions and organizations.
